# 架構決策紀錄

> 僅記錄影響後續開發方向的重大決策。
> Agent 提議 → 人類確認 → 才寫入。

---

## 2026-04-30
- 正式每日英文材料的主題輪替固定為 `daily → travel → daily → travel`。產出時必須依 `profile.json` 的 `lastTopic` 嚴格交替，不可因題材好寫或近期內容偏向而連續兩天使用同一主題。
- `daily` 題材不得長期收斂成居家清潔或房間整理；後續應優先挑選更貼近日常真實使用情境的場景，例如通勤、購物、點餐、排隊、問路、同事互動、家附近活動。
- 文章、片語、單字的選擇優先順序調整為「高頻日常口語可立即套用」高於「題材好湊」或「字面上能塞進文章」。

## 2026-04-09
- 正式每日英文學習模板改為專案內維護，路徑固定為 `.ai/daily-english-learning/SKILL.md`。後續規格調整優先修改這份本地模板，不再以專案外 `.claude/` 路徑為主。
- 單句播放 UI 採用「全域模式切換」而非每句獨立循環按鈕；預設模式為「單次」，使用者可切到「循環」後再點任一句重複播放。
- 每次新增 `daily/YYYY-MM-DD/` 的正式學習材料後，首頁 `index.html` 的學習記錄清單也必須同步加入當日條目。
- 首頁上方「前往今天的學習內容」按鈕會依日期自動導向，但這不代表下方學習記錄會自動更新；清單目前仍是手動維護。
